브라우저 객체란? (BOM, Browser Obejec Model)

브라우저에 내장된 객체를 '브라우저 객체'라고 한다.
window는 브라우저 객체의 최상위 객체이며 window 객체에는 하위 객체가 포함되어 있다.

open() 메서드

open("URL", "새창 이름", "새창 옵션")

alert()

alert("경고 메세지");

prompt()

prompt("질의 내용","기본답변");

confirm()

confirm("질의내용"); - 확인/취소

/* 생략

screen 객체

screen.속성 - 사용자의 모니터 정보(속성)을 제공하는 객체 - 너비 높이 컬러

location 객체

location.속성; location.메서드() -사용자 브라우저와 관련된 속성과 메서드
