Math.속성 또는 메소드

수학에서 사용하는 특수함수나 특수값을 사용할 수 있게 해준다.
특수함수란 지수함수, 삼각함수, 로그함수등을 말한다.

E 오일러 상수 약 2.718
LN10 10의 자연 로그 값 약 2.302
LN2 2의 자연 로그 값 약 0.693
LOG10E 밑이 10인 E 로그 약 0.434
LOG2E 밑이 2인 로그 약 1.442
PI 원주율 약 3.14159
SQRT1_2 1/2의 제곱근 SQRT{} 약 0.707
SQRT2 2의 제곱근 SQRT{} 약 1.414

복잡한 계산을 풀어주는 Math 메소드


max(x,y)    x와 y의 수 중에서 가장 큰 수
min(x,y)    x와 y의 수 중에서 가장 작은 수
round(x)    x를 소수첨 첫째 자리를 반올림한다
ceil(x)     소수점 자리를 무조건 올림
floor(x)     소수점 자리를 무조건 절삭
random()     0과 1사이 난수를 발생한다
abs(x)     수의 절대값을 반환
cos(x) x의 cosine 값을 구해준다
acos(x) cosine의 역함수(arc cosine)
asin(x) sin의 역함수(arc sin)
atan(x) tangent의 역함수(arc tan)
atan2(x,y) tangent의 역함수(arc tan)
exp(x) x의 지수 함수를 구해준다
log(x) x의 로그 함수를 구한다
pow(x,y) 지수 함수 f(x,y)=xy
sin(x) x의 sine 값을 구한다
sqrt(x) x의 제곱근을 구한다
tan(x) x의 tangent를 구한다

rondom 메서드를 이요해서 원하는 숫자(작은수~큰수) 사이의 난수값을 도출하는 공식

Math.floor(Math.random()*(큰수-작은수+1))+작은수

String(문자열) 객체

String객체가 내장 객체 중에서 활용 빈도가 가장 높을 것임

보편적인 생성법

var  변수(인스턴트 네임) = new String("자바스크립트");

간편한 생성법

var 변수(인스턴스 네임)="자바스크립트";

 

var theText="web js hi ciy";
theText.bold()    텍스트를 지한게 표현하여 반환 
theText.link('URL')    텍스트에 링크를 걸어 반환
theText.length    텍스트의 개수를 반환 (13)
theText.toLowerCse()    영문자를 소문자로 바꿔 변화
theText.toUpperCase()    영문자를 대문자로 바꿔 반환 
theText.indexOf("i")    처음부터 "i"를 찾아 최초로 일치하는 인덱스 번호를 반환, 없을 경우 -1을 반환 (8)
theText.lastIndexOf("i")    끝에서부터 "i"를 찾아 최초로 일치하는 인덱스 번호를 반환, 없을 경우 -1을 반환 (11)
theText.charAt(5)    5번째 인덱스에 저장된 문자 데이터를 반환(s)
theText.substring(4,6)    4번부터 6번 인덱스까지의 문자 데이터를 반환 (js)
theText.substr(10,3)    10번 인덱스부터의 3글자를 반환(ciy)
theText.replace("web","bye")    텍스트 중 "web"을 "bye"로 치환해서 반환(bye js hi ciy)
theText.slice(7,9)    7번부터 9번 인덱스까지의 일부만 잘라서 반환(hi)
theText.concat("good")    끝에 "good"이라는 문자를 결합하여 반환(~good)
theText.split("")    (공백)을 기준으로 데이터를 분리(콤마)(web,js,hi,ciy)
